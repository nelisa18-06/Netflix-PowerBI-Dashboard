# Netflix Content Analysis Dashboard (Power BI)

## Overview
This project analyzes Netflix content using Power BI to uncover trends in content growth, genre distribution, and content types.

## Key Insights
- Netflix content increased significantly after 2015  
- Movies make up around 70% of the platform  
- Drama and Documentary are the most common genres  

## Tools Used
- Power BI  
- DAX  
- Data Cleaning  

## Features
- Interactive dashboard with filters (Year, Country, Type)  
- KPI cards for insights  
- Clean dark-themed design
##  Goal
The goal of this project is to analyze Netflix content data and build an interactive dashboard that provides insights into content trends, distribution, and genre patterns. This project demonstrates skills in data cleaning, data modeling, DAX, and data visualization using Power BI.
